How backup and branch using Git and Github!

  • create a directory (mkdir 'name')
  • move into the directory (cd 'dir name')
  • initialize Git in your folder, initial commit (git init)
  • create your html, css, js files (touch 'filename')
  • git add . then git commit. This is yor initail commit
  • next create the remote repo (git remote add origin git@github.com:Muffinman75/branch-practice.git:Muffinman75/'folder name'.git
  • set up github repo to track local one (git push -u origin master, only git push there on after.)
  • work on your project, saving and pushing as you go, til a point is reached when you need to branch because the changes you wish to make are deep. Now it's time to branch!
